# UE5-Lyoko-V1
 Unreal Engine 5 ile geliştirlmiş 3. şahıs kameralı melee combat game. Masaüstü sürümler için geçerlidir.

# Unreal Engine 5 Showcase

### Oynanış Videosu
[![Oyun Gameplay Videosu](https://img.youtube.com/vi/TIo6Bse82rE/maxresdefault.jpg)](https://www.youtube.com/watch?v=TIo6Bse82rE)

*Projeyi ve mekanikleri detaylı görmek için yukarıdaki görsele tıklayarak videoyu izleyebilirsiniz.*

*Temel karakter sınıflarını, saldırı mekaniklerini, damage sistemini, kamera ayarlamalarını ve benzeri pek çok şeyi C++ ile yaptım. Geriye kalan UI, harita tasarımı input girişleri ve bazı can barı gibi widget özelliklerini BLueprint ile yaptım. C++ ile oluşturduğum karakter sınıflarını Unreal editörü içerisindeki oluşturulan Blueprint sınıflarına parent class olarak atadım ve değişiklikleri, ayrıntıları C++ da verdiğim özelliklerin detaylarını editörden ayarladım.*
